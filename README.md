# MuziJanpanese

## 日语构造
1. `あたま`
**平假名**  标注汉字，构成日语单词

2. `頭`
**日汉字**  
- 简体
-- yama    山
-- neko    猫
-- sennsei 先生             
- 繁体     
-- de nnwa     電話
-- he nn ta i  変態                   
- 和字     
- 片假名     标注外来语，拟声词   
-- a i su ku ri mu      アイスクリム

3. a ta ma **罗马音**  标注假名发音
